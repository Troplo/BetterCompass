# BetterCompass
BetterCompass uses a proxy for the Compass API to allow for a seamless integration with BetterCompass.<br>
This can be found in `/proxy`, and can be run with `node .`

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
